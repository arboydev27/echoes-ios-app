# Echoes

An iOS application built with Swift and SwiftUI.

## Overview

A simple iOS application built using SwiftUI. This project is a starting point for further development.

## Getting Started

1. Open `Echoes.xcodeproj` in Xcode.
2. Build and run the project using the iOS Simulator or a physical device.

## Requirements

- Xcode (latest recommended)
- iOS 16.0+ (or as specified in the project deployment target)
